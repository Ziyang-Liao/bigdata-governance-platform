# GOV-03 血缘可视化

> 优先级: P1 | 模块: 数据治理

## 1. 功能概述
交互式血缘图使用 ReactFlow 实现。节点=数据表（显示表名+数据源类型图标），边=数据流向。支持：展开/收起上下游N层、搜索定位表、点击表查看详情、列级血缘展开（点击表展开字段级连线）。前端：独立页面 /governance/lineage?table=xxx。从 /api/governance/lineage/{fqn} 获取上下游关系，递归加载。ReactFlow 自动布局（dagre 算法）。节点颜色区分数据源类型（MySQL蓝/S3绿/Redshift紫）。
