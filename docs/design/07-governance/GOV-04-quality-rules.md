# GOV-04 数据质量规则

> 优先级: P1 | 模块: 数据治理

## 1. 功能概述
配置数据质量检查规则：非空检查（指定字段不允许 NULL）、唯一性检查（指定字段值唯一）、范围检查（数值在 min-max 范围内）、正则检查（字段值匹配正则表达式）、自定义 SQL 检查（返回 0 表示通过）。定时执行检查，生成质量报告。DynamoDB bgp-quality-rules 表。前端：规则列表+新建规则弹窗（选择表→选择字段→选择规则类型→配置参数）。质量报告页面：通过率、异常数据样本。后端：EventBridge 定时触发 Lambda，对每条规则生成检查 SQL 并在 Redshift 执行，结果写入 bgp-quality-reports。
