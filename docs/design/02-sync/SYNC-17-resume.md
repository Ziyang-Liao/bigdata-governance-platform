# SYNC-17 断点续传

> 优先级: P2 | 模块: 数据同步

## 1. 功能概述
增量同步失败后从上次水位线继续，不重复同步已完成的数据。依赖 SYNC-05 水位线管理。实现：同步失败时不更新水位线，下次运行自动从上次水位线开始。全量同步失败时：记录已完成的表，下次跳过已完成的表。前端：运行历史中显示"从断点继续"按钮。后端：检查上次运行状态，如果失败则从 bgp-watermarks 读取上次水位线继续。
