# MON-03 告警规则配置

> 优先级: P0 | 模块: 任务监控

## 1. 功能概述
配置告警规则：触发条件（任务失败/超时/数据量异常）+ 适用范围（所有任务/指定任务）+ 通知渠道。DynamoDB 新表 bgp-alert-rules（PK: userId, SK: ruleId）。字段：name, condition:{type,threshold}, taskScope:[taskIds], channels:[{type,target}], enabled。前端：告警规则列表页，新建/编辑规则弹窗。条件类型：task_failed（任务失败时）、task_timeout（超过N分钟）、row_count_anomaly（行数偏差超过N%）。后端：任务完成时检查匹配的告警规则，触发通知。
