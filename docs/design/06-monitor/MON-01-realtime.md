# MON-01 实时刷新

> 优先级: P0 | 模块: 任务监控

## 1. 功能概述
监控页面自动轮询刷新任务状态。支持配置刷新间隔（5s/10s/30s/关闭）。运行中的任务状态实时更新，无需手动刷新。前端：useEffect + setInterval 轮询 /api/sync 和 /api/workflow。右上角刷新间隔选择器。运行中任务行高亮闪烁。状态变更时 message.info 提示。优化：仅请求 status 变更的增量数据（If-Modified-Since 或 lastUpdatedAt 过滤）。
