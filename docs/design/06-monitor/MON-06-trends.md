# MON-06 运行统计趋势

> 优先级: P1 | 模块: 任务监控

## 1. 功能概述
Dashboard 展示最近 7/30 天的统计趋势：任务成功率折线图、平均耗时折线图、每日同步数据量柱状图、失败任务 Top 5。前端：使用 @ant-design/charts 绑制图表。数据来源：聚合 bgp-task-runs 表。时间范围切换（7天/30天/自定义）。后端：GET /api/monitor/trends?days=7 返回按天聚合的统计数据 [{date, totalRuns, successRuns, failedRuns, avgDuration, totalRows}]。
