# SCH-02 调度日历视图

> 优先级: P1 | 模块: 调度管理

## 1. 功能概述
日历形式展示所有任务的调度计划。支持日/周/月视图切换。每个时间格显示该时段要执行的任务列表。直观查看任务分布和时间冲突。前端：使用 Ant Design Calendar 或 FullCalendar 组件。从所有任务的 cronExpression 计算未来 30 天的执行计划。任务以彩色标签显示在日历格中。点击任务跳转到详情页。后端：GET /api/schedule/calendar?start=2026-03-01&end=2026-03-31 返回时间范围内所有计划执行。

## 2. 用户故事
