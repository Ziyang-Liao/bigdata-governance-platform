# SCH-01 可视化 Cron 编辑器

> 优先级: P0 | 模块: 调度管理

## 1. 功能概述
图形化 Cron 编辑器替代手动输入。选择调度类型（分钟/小时/每天/每周/每月），图形化选择执行时间。实时预览下次 5 次执行时间。前端：独立组件 CronEditor，包含类型切换 Radio + 时间选择器 + Cron 表达式显示 + 预览列表。类型切换时自动生成 Cron：每天→选时分，每周→选周几+时分，每月→选日期+时分。预览使用 cron-parser 库计算下次执行时间。后端：无需后端，纯前端组件。Cron 表达式存入 DynamoDB。
