# WF-05 DAG 校验

> 优先级: P1 | 模块: ETL编排

## 1. 功能概述
保存/发布前自动校验 DAG：环检测（拓扑排序，有环则高亮环路节点）、孤立节点检测（无入边也无出边的节点）、必填参数检查（SQL 节点必须有 SQL、同步节点必须关联任务）、连线合法性（条件节点必须有 true/false 两条出边）。前端：校验失败时高亮问题节点（红色边框+错误提示 tooltip），底部显示错误列表。后端：纯前端校验（图算法），不需要 API。使用 Kahn 算法检测环，DFS 检测连通性。
