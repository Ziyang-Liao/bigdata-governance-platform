# AUTH-03 用户管理页面

> 优先级: P0 | 模块: 用户权限

## 1. 功能概述
管理员专属页面 /admin/users。功能：用户列表（用户名/邮箱/角色/状态/创建时间）、创建用户（AdminCreateUser）、分配角色（AdminAddUserToGroup）、禁用/启用用户（AdminDisableUser/AdminEnableUser）、重置密码（AdminSetUserPassword）。前端：用户列表表格+新建用户弹窗+角色分配下拉。仅 Admin 角色可访问。后端：/api/admin/users CRUD，调用 Cognito Admin API。
