# AUTH-04 全局操作审计

> 优先级: P0 | 模块: 用户权限

## 1. 功能概述
记录所有用户操作到审计日志。复用 DS-18 审计日志方案，扩展到所有模块（数据源/同步/工作流/Redshift/调度/用户管理）。全局审计中间件：在每个写操作（POST/PUT/DELETE）的 API handler 中自动记录。审计记录：userId, action(create/update/delete/execute), resourceType, resourceId, timestamp, details(变更内容), ip。前端：/admin/audit 审计日志页面，支持按用户/资源类型/操作类型/时间范围过滤。仅 Admin 可查看完整审计日志，Developer 可查看自己的操作记录。
