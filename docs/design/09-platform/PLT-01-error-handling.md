# PLT-01 统一错误处理

> 优先级: P0 | 模块: 平台基础

## 1. 功能概述
API 统一返回格式：{success: boolean, data: any, error: {code: string, message: string}}。HTTP 状态码规范：200成功/400参数错误/401未认证/403无权限/404不存在/500服务器错误。前端全局错误拦截：fetch wrapper 统一处理错误响应，message.error 显示错误信息。网络异常自动重试（最多 3 次，指数退避）。后端：创建 apiResponse(data) 和 apiError(code, message, status) 工具函数，所有 API handler 使用。全局 try-catch 包装，未捕获异常返回 500。
