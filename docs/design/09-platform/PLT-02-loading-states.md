# PLT-02 加载状态优化

> 优先级: P0 | 模块: 平台基础

## 1. 功能概述
骨架屏（Skeleton）：页面首次加载时显示骨架屏而非空白。按钮 loading 状态：提交/保存/删除等操作时按钮显示 loading 并禁用，防止重复提交。长操作进度条：同步任务启动、工作流发布等耗时操作显示进度提示。前端：封装 useLoading hook，管理 loading 状态。Ant Design Skeleton 组件用于列表/详情页。Button loading prop 绑定操作状态。NProgress 顶部进度条用于页面切换。
